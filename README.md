# <h1 align= "center">Proyecto de Videos de YouTube Favoritos</h1>

<h3> 🛠 Este Proyecto fue desarrollado con el Stack MERN con TypeScript tanto en el backend como en el frontend</h3>

<p aligth="justify">El proyecto fue creado con el fin de probar el uso de TypeScript tanto en el backend como con el frontend y ver cuáles son las ventajas que se obtienen al desarrollar
  el proyecto bajo el tipado de datos que proporcionado TypeScript.</p>
  
<hr>

[![proyecto-typescript.png](https://i.postimg.cc/YSrqqFXB/proyecto-typescript.png)](https://postimg.cc/HrNC6rD6)

<hr>

### <h3> :computer: Este Proyecto cuenta con las siguientes funciones: </h3>


<p aligth="justify"> ⭐️ Función de Crear Videos Favoritos: Se podrá ingresar información básica como Título, Descripción y url del video en Youtube. </p>

<p aligth="justify"> ⭐️ Función de Listar Videos Favoritos: Se podrá ver en tarjetas cada uno de los videos favoritos agregados. </p>

<p aligth="justify"> ⭐️ Función de Actualizar un Video: Se podrá actualizar la información de título, descripción y url del video seleccionado.</p>

<p aligth="justify"> ⭐️ Función de Eliminar Video: Se podrá eliminar un video seleccionado de la lista de Videos registrados.</p>


</br>


### <h3> ⚡ Instalación de dependencias tanto para los directorios backend y frontend: </h3>
- npm install

### <h3> ⚡ Para ejecutar el directorio backend se debe aplicar el siguiente comando </h3>
- npm run dev

### <h3> ⚡ Para ejecutar el directorio frontend se debe aplicar el siguiente comando </h3>
- npm start
